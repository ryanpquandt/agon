# Drafts

The manuscript lives here. **These files are the human's** — Agon marks them up,
questions them, and (in nonfiction, on request) proposes options, but does not
author or overwrite them. The one exception: a consented `ai-draft` in a
technical-delegable project, which is named and logged as such.

Naming follows the draft cycle (see `../../../method/draft-cycles.md`):

- `draft-01.md`, `draft-02.md`, … — **full drafts**, one per writing-room
  cycle. Immutable: never overwritten, so the record shows the piece's whole
  trajectory (including drafts abandoned by a `redirect`).
- After the altitude decision is `descend`, the polish funnel adds the layer
  passes (see `../../../method/layered-editing.md`):
  - `d1-structure.md` — after the structure pass
  - `d2-paragraph.md` — after the paragraph pass
  - `d3-sentence.md` — after the sentence pass
  - `d4-word.md` — after the word pass → the final manuscript

Every full draft gets a Ledger entry (`draft: <desc> — id: draft-NN`) — the
engine will not close a review without one, and will not open the polish funnel
without a `descend` decision.
